### Exercício 1
[215. Kth Largest Element in an Array](https://leetcode.com/problems/kth-largest-element-in-an-array/description/)
- Dificuldade: Médio
- Dado um array de números inteiros nums e um inteiro k, retorne o k-ésimo maior elemento em nums.
- Observação: É o k-ésimo maior elemento na ordem classificada, não o k-ésimo distinto.


### Exercício 2
[23. Merge k Sorted Lists](https://leetcode.com/problems/merge-k-sorted-lists/description/)
- Dificuldade: Difícil
- É passado uma array de n listas encadeadas, cada lista encadeada está ordenada em ordem crescente.
- Objetivo: Mesclar todas as listas encadeadas em uma única lista encadeada ordenada e retornar ela.
